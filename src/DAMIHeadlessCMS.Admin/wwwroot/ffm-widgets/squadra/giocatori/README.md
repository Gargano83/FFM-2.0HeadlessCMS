Cartella di destinazione del bundle compilato del componente Angular
"Database Giocatori" (FFM.Giocatori).

Il file `main.js` (bundle singolo, generato con `npm run build:giocatori-widget`
dal progetto Angular `syncfusion-tfl-app`) va copiato qui prima di pubblicare
DAMIHeadlessCMS.Admin. Non è generato automaticamente dalla build .NET: è un
artefatto Angular separato, gestito nel suo repository/pipeline dedicato.

Vedi il README del progetto Angular per le istruzioni di build, e
docs/ROADMAP.md (fase 7.1) per il contesto architetturale.
